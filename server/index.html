Rails Girls Guides

מטרת המדריכים האלו היא לתת לנשים כלים וקהילה כדי להבין טכנולוגיה ולבנות את הרעיונות שלהן. ארגני ארועים משלך, הוסיפי מדריכים חדשים, או פשוט השתמשי בהם כדי ללמוד Rails. לעוד, הוסיפי את עצמך לרשימת התפוצה של הצוות.
